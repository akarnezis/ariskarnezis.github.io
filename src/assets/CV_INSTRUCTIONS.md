# CV Placeholder

To add your CV to this website:

1. Create or export your CV as a PDF file
2. Name it `cv.pdf`
3. Place it in the `/assets` directory
4. The "Download CV" button in the Hero section will automatically link to this file

The download link is configured as:
- Path: `/assets/cv.pdf`
- Download filename: `Aris_Karnezis_CV.pdf`

You can update the download filename in `/components/Hero.tsx` if needed.
